# Student Database Project

This is a simple project using Flask and MySQL.

It displays student records from a database on a webpage.

## Features
- Connects Python (Flask) with MySQL
- Displays student data in a table
- Simple and easy to understand

## Technologies Used
- Python (Flask)
- MySQL
- HTML & Bootstrap

## How to Run
1. Set your database password in environment variable
2. Run:
   python app.py
3. Open:
   http://127.0.0.1:5000
